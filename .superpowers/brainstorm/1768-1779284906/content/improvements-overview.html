三项改进点概览

自动持久化 + 树导航流畅度 + Markdown 渲染增强

改进 1: 自动持久化

页面刷新后树数据自动恢复，Toolbar 顶部显示保存状态指示器。

Toolbar 顶部
🌳 JavaScript 闭包学习 自动已保存 · 12:03

改动

  • Store subscribe → localStorage 防抖写入
  • 启动时优先读取 localStorage，无则用示例数据
  • ~5MB 上限时给出提示

改进 2: 树视图交互增强

选中节点时树自动滚动到视野内；方向键切换选中节点；展开/折叠带过渡动画。

Before: 节点在视野外

↳ [子1] 闭包的应用场景？
↳ [子2] 闭包在实际开发中？
↳ [孙] 代码示例？
🎯 选中但不可见
After: 自动滚动到选中节点
🎯 自动滚到这里
↳ [子2] 闭包在实际开发中？
↳ [孙] 代码示例？

改进 3: 快捷键 & 引导

方向键导航，按 ? 显示快捷键面板。

快捷键提示面板
↑ ↓切换选中节点
← →展开/折叠节点
Enter聚焦输入框
Ctrl+Enter发送提问
E编辑当前节点
Delete删除当前节点

改进 4: 展开/折叠动画

子树展开折叠使用 CSS transition — max-height + opacity 过渡，而不是瞬间消失。

Before: 瞬间变化
[父节点] 点击折叠
⬇ 子树直接消失 ⬇
After: 250ms 平滑收起
[父节点] 点击折叠
⬇ max-height → 0, opacity → 0 ⬇
